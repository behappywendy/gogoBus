# GOGO BUS

## 介紹:
- 顯示附近800公尺或1000公尺站點資訊
- 查詢各縣市公車路線
- 查詢各縣市站點資訊
- 回報到站時間
- 公車最新消息
- 地圖顯示站牌位置
- 具有夜間模式

## 使用技術
* vue cli
* axios
* vuex
* Vue3 OpenLayers

## 第三方服務
- ptxAPI
